## 项目架构

本项目包含两个作业，采用模块化设计，主要包含以下文件：

### 作业 1：三角形 3D 变换
- `src/Work1/main_1.py`：项目主入口，实现三角形的 3D 变换和渲染

### 作业 2：立方体 3D 变换
- `src/Work1/main_2.py`：项目主入口，实现立方体的 3D 变换和渲染

## 代码逻辑

### 作业 1：三角形 3D 变换

1. **初始化**：
   - 使用 Taichi 库初始化 CPU 环境
   - 声明顶点和屏幕坐标的存储字段
   - 初始化三角形的三个顶点坐标

2. **变换矩阵计算**：
   - 模型变换：绕 Z 轴旋转
   - 视图变换：将相机移动到原点
   - 投影变换：透视投影
   - 组合 MVP 矩阵

3. **顶点变换**：
   - 对每个顶点应用 MVP 变换
   - 执行透视除法，转换为 NDC 坐标
   - 视口变换，映射到屏幕空间

4. **渲染**：
   - 创建 GUI 窗口
   - 绘制三角形的边
   - 响应用户的键盘输入（A/D 键旋转）

### 作业 2：立方体 3D 变换

1. **初始化**：
   - 使用 Taichi 库初始化 OpenGL 后端（利用核显提高性能）
   - 声明顶点和屏幕坐标的存储字段
   - 初始化立方体的 8 个顶点坐标

2. **变换矩阵计算**：
   - 模型变换：绕 X 轴和 Y 轴旋转
   - 视图变换：将相机移动到原点
   - 投影变换：透视投影
   - 组合 MVP 矩阵

3. **顶点变换**：
   - 对每个顶点应用 MVP 变换
   - 执行透视除法，转换为 NDC 坐标
   - 视口变换，映射到屏幕空间

4. **渲染**：
   - 创建 GUI 窗口
   - 绘制立方体的边（使用不同颜色）
   - 响应用户的键盘输入（A/D/W/S 键旋转）

## 实现功能

### 作业 1：三角形 3D 变换
- **3D 变换**：实现了三角形的 3D 旋转效果
- **透视投影**：使用透视投影矩阵将 3D 坐标转换为 2D 屏幕坐标
- **用户交互**：通过 A/D 键控制旋转角度
- **实时渲染**：实时显示变换后的三角形

### 作业 2：立方体 3D 变换
- **3D 变换**：实现了立方体的 3D 旋转效果（支持双轴旋转）
- **透视投影**：使用透视投影矩阵将 3D 坐标转换为 2D 屏幕坐标
- **彩色线框**：为立方体的不同边设置不同颜色
- **GPU 加速**：使用 OpenGL 后端，利用核显提高性能
- **用户交互**：通过 A/D/W/S 键控制旋转角度
- **实时渲染**：实时显示变换后的立方体

## 运行方式

### 作业 1：三角形 3D 变换
在项目根目录下运行：

```bash
python src/Work1/main_1.py
```

运行后会弹出一个窗口，按 A/D 键可以控制三角形绕 Z 轴旋转。

### 作业 2：立方体 3D 变换
在项目根目录下运行：

```bash
python src/Work1/main_2.py
```

运行后会弹出一个窗口，按 A/D 键可以控制立方体绕 Y 轴旋转，按 W/S 键可以控制立方体绕 X 轴旋转。
